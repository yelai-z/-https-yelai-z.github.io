# nrm管理npm源

  1. 安装
npm install nrm -g
  1. 如果中途报错：
Please try running this command again as root/Administrator.

那么说明缺少权限，解决方法，将owner变成自己：

sudo chown -R $USER /usr/local
  1. 查看帮助
nrm --help
  1. 添加新源：
nrm add newreg http://registry.npmjs.new.com
  1. 展示所有可用源
nrm ls
  1. 测试每个源的速度
nrm test
  1. 选择源
nrm use cnpm
